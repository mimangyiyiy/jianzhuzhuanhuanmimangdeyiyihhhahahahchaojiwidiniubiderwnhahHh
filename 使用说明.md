# MC 建筑转换器 v3.6 — 使用说明

## 简介

MC 建筑转换器是一款纯前端 Minecraft 建筑指令生成工具。将 `.schematic` 或 `.litematic` 建筑文件转换为基岩版 `/setblock` / `/fill` 指令，支持 unless 反选模式（2 个盔甲架自动扫描放置），内置 24 条网易版敏感词规则。

**无需安装任何软件**，浏览器打开即用。

---

## 快速开始

### 在线使用（推荐）

1. 访问部署网址（如 CloudFlare Pages 提供的域名）
2. 等待 Pyodide 加载（首次约 20-30 秒）
3. 看到 "已就绪" 后，拖放或点击选择建筑文件
4. 转换完成后自动下载指令文件

### 本地使用

```bash
# 方法 1: 用 Python 内置服务器
cd 网页端
python -m http.server 8080
# 浏览器打开 http://localhost:8080

# 方法 2: 用 Node.js
npx http-server 网页端 -p 8080
```

## 功能说明

| 功能 | 说明 |
|------|------|
| 文件格式 | `.schematic` (WorldEdit)、`.litematic` (Litematica) |
| 指令模式 | unless 实体反选（2 盔甲架自动扫描） |
| 方块压缩 | 2D Fill 压缩（同行同方块合并为 fill 指令） |
| 敏感词 | 24 条正则规则，含敏感方块自动拆分手动 setblock |
| 输出 | `文件名_commands.txt`，含完整免责声明 |

### 指令使用

生成的指令按顺序放入命令方块：
1. **脉冲**命令块：计分板初始化 + 召唤盔甲架
2. **循环**命令块：`scoreboard players add ...`
3. **连锁**命令块（全部）：unless 放置指令 + 坐标切换

末尾 `[手动执行]` 段的 setblock 指令需在游戏内手动逐条执行。

---

## CloudFlare Pages 部署

1. 登录 [CloudFlare](https://dash.cloudflare.com) → Workers 和 Pages → Pages
2. 创建应用程序 → 直接上传
3. 将 `网页端` 文件夹拖入上传区
4. 点击部署，获取 `xxx.pages.dev` 域名

---

## 技术架构

- **Pyodide**（CPython 3.11 → WebAssembly）：在浏览器中运行完整 Python
- **NBT 解析**：自研 NBTReader，无需 nbtlib 依赖
- **方块映射**：687 BEDROCK_ID_MAP + 138 LEGACY 别名 = 825 条
- **敏感词**：24 条正则规则（网易版合规）
